## 作业1 最大间隔问题
### 问题描述

最大间隙问题。给定 n 个实数x1, x2,…,xn，求这n个实数在数轴上相邻2个数之间的最大差值。
假设对任何实数的下取整函数耗时O(1), 设计解最大间隙问题的线性时间算法。

解决思路：鸽舍原理
